## Quick Examples

### Generate Customers
```python
from data_designer.essentials import (
    DataDesigner,
    DataDesignerConfigBuilder,
    SamplerColumnConfig,
    SamplerType,
    PersonFromFakerSamplerParams,
)

data_designer = DataDesigner()
config = DataDesignerConfigBuilder()

config.add_column(
    SamplerColumnConfig(
        name="customer",
        sampler_type=SamplerType.PERSON_FROM_FAKER,
        params=PersonFromFakerSamplerParams(
            include_age=True,
            include_gender=True,
            include_email=True,
        ),
    )
)

result = data_designer.design(config_builder=config, num_records=100)
df = result.dataframe
```

### Generate Products with LLM Descriptions
```python
from data_designer.essentials import (
    CategorySamplerParams,
    LLMTextColumnConfig,
    ModelConfig,
)

config.add_column(
    SamplerColumnConfig(
        name="category",
        sampler_type=SamplerType.CATEGORY,
        params=CategorySamplerParams(
            values=["Electronics", "Clothing", "Home"],
        ),
    )
)

config.add_column(
    LLMTextColumnConfig(
        name="product_name",
        model_alias="nvidia-text",
        prompt="Generate a catchy product name for {{ category }}",
    )
)
```

### Generate Campaign Performance
```python
config.add_column(
    SamplerColumnConfig(
        name="campaign_type",
        sampler_type=SamplerType.CATEGORY,
        params=CategorySamplerParams(
            values=["Newsletter", "Promotional", "Abandoned Cart"],
        ),
    )
)

config.add_column(
    SamplerColumnConfig(
        name="open_rate",
        sampler_type=SamplerType.GAUSSIAN,
        params=GaussianSamplerParams(
            mean=0.22,
            std_dev=0.05,
            min_value=0.10,
            max_value=0.40,
        ),
    )
)
```
